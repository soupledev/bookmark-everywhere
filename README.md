Bookmark Everywhere
---

This chromium extension is inspired from how Safari shows bookmarks:
- Show bookmark gallery in a grid layout like for safari on a new tab
- Open bookmark gallery everywhere with Ctrl/Cmd-Shift-K shortcut
- Keyboard-centric: you can navigate the bookmarks with arrow, escape to close or move to the parent folder

Main features:
- Very fast: all data is cached to have a snappy experience
- Website favicon with a good quality and not the tiny default 16x16 or 32x32 ones
